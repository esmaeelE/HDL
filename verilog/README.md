# verilog

```
sudo apt install iverilog verilator gtkwave
```

# Run

Compile
```
iverilog -o and_gate_tb and_gate.v and_gate_tb.v
```

```
vvp and_gate_tb
```

```
gtkwave and_gate.vcd
```
